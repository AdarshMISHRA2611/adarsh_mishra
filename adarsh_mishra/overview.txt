CABBY is a browser-based online gaming portal where users can discover and play free HTML5 games without downloading anything. The website has four pages: a home page with a daily missions section above a searchable grid of game cards filterable by category; a game page that loads any selected game inside an iframe and shows the user's personal best score for that game; a leaderboard page with two tabs ("All Scores" and "Your Best") that tracks saved scores; and a profile page where users can set a display name, manage favourite games and saved bookmarks, view stats including their current and best streak. Every day, all users see the same 3 missions. Completing all 3 missions in a day increments the user's streak counter; missing a day resets the current streak to zero. The current streak is shown on the navigation bar of every page. Users can mark games as favourites and save bookmarks during play to resume later. Their highest score per game is tracked automatically. The entire site runs on the client side with no backend, no database, and no API keys of any kind. All user data is stored in the browser using localStorage. Games are embedded from freely available HTML5 sources; the freelancer only builds the website shell that hosts them. The deliverable is a complete, responsive portal delivered as source code on GitHub plus a live deployed link.
